## ⚠️ 常见问题

### 问题1：看不到 "Run workflow" 按钮

**原因：** 权限未开启

**解决：**
1. 进入 Settings → Actions → General
2. 选择 "Read and write permissions"
3. 保存后刷新页面

### 问题2：按钮是灰色的点不了

**原因：** 没有权限或代码未上传

**解决：**
1. 确保你是仓库的拥有者
2. 确保代码已推送到GitHub

### 问题3：运行失败（红色✗）

**原因：** 可能是网络问题或代码错误

**解决：**
1. 点击查看失败详情
2. 看是哪个步骤出错
3. 重试运行

---

**简单总结：** 点 Actions → 点 VC Tracker Deploy → 点 Run workflow → 等绿色✓ → 完成！
